\subsection{GitHub}\label{appendix:GitHub}

During this project I have stored all my data and produced code in a public GitHub repository. The repository is named "optomechanics" after the group at the Department of Physics and Astronomi at Aarhus University in which I have made this Master's project.

Links/terminal commands to clone repository: 
\begin{itemize}
    \item HTTPS: https://github.com/Mikkelodeonn/optomechanics.git 
    \item SSH: git@github.com:Mikkelodeonn/optomechanics.git
    \item GitHub CLI: gh repo clone Mikkelodeonn/optomechanics
\end{itemize}
